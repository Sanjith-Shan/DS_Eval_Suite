The audit of `/workspace/pipeline.py` revealed several critical sources of data leakage that resulted in an unrealistically high test accuracy of ~97%:

1.  **Scaling before split:** `StandardScaler` was fit on the entire dataset, leaking the global mean and variance into the training process.
2.  **Target Encoding before split:** High-cardinality categorical features (like `customer_segment`) were encoded using the target values of all rows, including those in the test set. Since `customer_segment` is nearly unique, this essentially mapped each row to its target.
3.  **Feature Selection before split:** `mutual_info_classif` was used on the entire dataset to select the top features, which uses the relationship between features and targets in the test set to guide the training phase.
4.  **Late Splitting:** The `train_test_split` was performed after all the above transformations, ensuring that the "test" data was already contaminated by information it was supposed to be isolated from.

The corrected pipeline in `/output/pipeline_fixed.py` addresses these issues by:
- Splitting the data into training and test sets **before** any transformations.
- Fitting the `StandardScaler` only on the training set and using it to transform both sets.
- Computing target encoding means only from the training set and handling unseen categories in the test set by using the global training target mean.
- Performing feature selection (`mutual_info_classif`) solely on the training set.

The corrected pipeline now reports a realistic test accuracy of **~68.1%**.

```python
"""Customer churn prediction pipeline (FIXED).

This version addresses data leakage by:
1. Splitting data before any transformations.
2. Fitting scaler, target encoder, and feature selection ONLY on the training set.
3. Correctly handling unseen categories in the test set.
"""

import numpy as np
import pandas as pd
from sklearn.ensemble import GradientBoostingClassifier
from sklearn.feature_selection import mutual_info_classif
from sklearn.metrics import accuracy_score
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler


def train_and_evaluate(data_path: str = "/workspace/data.csv") -> float:
    df = pd.read_csv(data_path)
    y = df.pop("target")

    # Train / test split MUST happen before any data-dependent transformations.
    X_train, X_test, y_train, y_test = train_test_split(
        df, y, test_size=0.2, random_state=42, stratify=y
    )
    
    X_train = X_train.copy()
    X_test = X_test.copy()

    # 1. Scale numeric features.
    numeric_cols = X_train.select_dtypes(include=[np.number]).columns.tolist()
    scaler = StandardScaler()
    X_train[numeric_cols] = scaler.fit_transform(X_train[numeric_cols])
    X_test[numeric_cols] = scaler.transform(X_test[numeric_cols])

    # 2. Target-encode categoricals.
    object_cols = X_train.select_dtypes(include="object").columns.tolist()
    for col in object_cols:
        train_target_mean = y_train.mean()
        means = y_train.groupby(X_train[col]).mean()
        
        X_train[col + "_te"] = X_train[col].map(means)
        X_test[col + "_te"] = X_test[col].map(means).fillna(train_target_mean)
        
        X_train = X_train.drop(columns=[col])
        X_test = X_test.drop(columns=[col])

    # 3. Feature selection using ONLY training data.
    mi = mutual_info_classif(X_train, y_train, random_state=42)
    top_idx = np.argsort(mi)[-10:]
    
    selected_cols = X_train.columns[top_idx]
    X_train = X_train[selected_cols]
    X_test = X_test[selected_cols]

    # 4. Train the model.
    model = GradientBoostingClassifier(random_state=42)
    model.fit(X_train, y_train)
    
    # 5. Evaluate.
    y_pred = model.predict(X_test)
    acc = accuracy_score(y_test, y_pred)
    
    return float(acc)


if __name__ == "__main__":
    acc = train_and_evaluate()
    print(f"TEST_ACCURACY={acc:.4f}")
```
