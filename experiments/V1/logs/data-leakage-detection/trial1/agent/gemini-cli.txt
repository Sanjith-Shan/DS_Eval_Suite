The audit of `/workspace/pipeline.py` revealed three major sources of data leakage between the training and test sets:
1.  **Scaling:** `StandardScaler` was fit on the entire dataset, allowing the mean and standard deviation of the test set to influence the training features.
2.  **Target Encoding:** Categorical features were encoded using the target means calculated across the entire dataset, directly leaking label information from the test set into the training features.
3.  **Feature Selection:** `mutual_info_classif` was performed on the full dataset, selecting features that had a strong relationship with the target in both the train and test sets.

I have corrected these issues in `/output/pipeline_fixed.py` by:
-   Moving the `train_test_split` to the beginning of the pipeline.
-   Fitting the `StandardScaler` only on the training data.
-   Calculating target encoding means using only the training labels and handling unseen categories in the test set using the global training mean.
-   Performing feature selection based solely on the training data.

These changes resulted in a more realistic test accuracy of **0.6810**, compared to the original, inflated accuracy of **0.9690**.

### Fixed Pipeline Implementation (`/output/pipeline_fixed.py`)

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import GradientBoostingClassifier
from sklearn.feature_selection import mutual_info_classif
from sklearn.metrics import accuracy_score
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler


def train_and_evaluate(data_path: str = "/workspace/data.csv") -> float:
    df = pd.read_csv(data_path)
    y = df.pop("target")

    # Train / test split FIRST to avoid leakage between sets.
    X_train, X_test, y_train, y_test = train_test_split(
        df, y, test_size=0.2, random_state=42, stratify=y
    )

    X_train = X_train.copy()
    X_test = X_test.copy()

    # Scale numeric features using only training data statistics.
    numeric_cols = X_train.select_dtypes(include=[np.number]).columns.tolist()
    scaler = StandardScaler()
    X_train[numeric_cols] = scaler.fit_transform(X_train[numeric_cols])
    X_test[numeric_cols] = scaler.transform(X_test[numeric_cols])

    # Target-encode categoricals using ONLY training data to avoid leakage.
    categorical_cols = X_train.select_dtypes(include="object").columns
    for col in categorical_cols:
        # Calculate means on training data only.
        means = y_train.groupby(X_train[col]).mean()
        global_mean = y_train.mean()
        
        # Apply to training data.
        X_train[col + "_te"] = X_train[col].map(means)
        
        # Apply to test data, filling missing categories with training global mean.
        X_test[col + "_te"] = X_test[col].map(means).fillna(global_mean)
        
        # Drop original categorical columns.
        X_train = X_train.drop(columns=[col])
        X_test = X_test.drop(columns=[col])

    # Pick the most informative features using ONLY training data.
    mi = mutual_info_classif(X_train, y_train, random_state=42)
    top_idx = np.argsort(mi)[-10:]
    selected_features = X_train.columns[top_idx]
    
    X_train = X_train[selected_features]
    X_test = X_test[selected_features]

    # Train model.
    model = GradientBoostingClassifier(random_state=42)
    model.fit(X_train, y_train)
    
    # Evaluate on held-out test set.
    accuracy = accuracy_score(y_test, model.predict(X_test))
    return float(accuracy)


if __name__ == "__main__":
    acc = train_and_evaluate()
    print(f"TEST_ACCURACY={acc:.4f}")
```
